# EasyEvent Support Platform

Plateforme de customer support IA multi-applications avec escalade humaine automatique intégrée à l'écosystème EasyEvent.

---

## Fonctionnalités

| Fonctionnalité | Statut |
|----------------|--------|
| Widget chatbot intégrable |
| Chatbot IA (Grok xAI / Mock FAQ) 
| Escalade Discord automatique |
| Escalade Email automatique |
| Interface admin React pour agents |
| Base de connaissance (FAQ) | 
| Upload de documents (PDF, TXT, Images) |
| OCR pour images (Tesseract) | 
| Mode clair/sombre |
| Design responsive (web + mobile) | 
| Authentification admin |
| Historique des conversations |
| Intégration avec EasyEvent (Laravel) |
| Notifications Filament (Laravel) |
