================================================================================
                          RÉSUMÉ DES ENDPOINTS API
================================================================================

| Méthode | Endpoint | Description |
|---------|----------|-------------|
| POST    | /api/message/                    | Envoyer un message (client → bot/agent) |
| GET     | /api/conversation/<id>/          | Récupérer l'historique d'une conversation |
| POST    | /api/agent/reply/                | Agent répond à une conversation |
| GET     | /admin/conversations/            | Liste des conversations (admin/widget) |
| POST    | /admin/conversations/<id>/       | Répondre à une conversation (admin) |
